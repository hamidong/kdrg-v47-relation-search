[수정] app/main_window.py  (1,394줄 → ~1,580줄)
  ── 구조 변경 ──
  MainWindow.__init__():
    · resize(1500, 900) → resize(1700, 960)
    · selected_card: Optional[ResultCard] = None  추가
    · _result_cards: List[ResultCard] = []         추가
    · _setup_shortcuts() 호출 추가
    · _update_status_bar() 호출 추가
    · _restore_settings() 호출 추가 (창 복원은 마지막)

  _build_ui():
    · splitter를 self.main_splitter로 저장 (QSettings 접근용)
    · _build_notice() 호출 제거 (QStatusBar로 대체)

  _build_header():
    · 타이틀: "KDRG V4.7 코드 관계 검색기" (V4.7 추가)
    · 우측 단일 VersionBadge → VersionLabel/DataVersionLabel/DataScopeLabel 3개
    · 정보 버튼(InfoButton) 추가 → open_about_dialog() 연결
    · 검색 초기화 버튼(SearchResetButton) 추가 → _reset_search() 연결
    · 안내 배너: QLabel → AdvancedCautionBanner(QFrame) + CautionIcon 조합

  _render_results():
    · selected_card = None 초기화
    · _result_cards = [] 초기화
    · 카드 생성 시 _result_cards에 누적

  select_result():
    · 이전 카드 ObjectName "ResultCard" 복원 (QSS 강조 해제)
    · 새 카드 ObjectName "ResultCardSelected" 설정 (QSS 강조 적용)
    · unpolish/polish 호출로 즉시 스타일 재적용

  _build_condition_table_row():
    · code_type → 역할명 매핑 (_CODE_TYPE_ROLE 딕셔너리)
      상병코드 → 주진단 / 기타진단코드 → 기타진단
      수술·처치코드 → 수술·처치 / 검사·처치코드 → 검사·처치 / 부가코드 → 부가코드
    · exclusion=True 시 역할명 → "제외조건" (ExcludeRoleBadge)
    · CodeTableFrame 생성 후 populate_table 직접 호출 제거 (지연 채우기)
    · toggle_expanded: checked=True일 때 ensure_populated() 호출

  ── 신규 메서드 ──
  _setup_shortcuts():
    Ctrl+L → search_edit 포커스 + 전체선택
    Esc    → 검색어 있으면 초기화, 없으면 search_edit 포커스

  _handle_esc():
    _setup_shortcuts의 Esc 처리 분리

  _reset_search():
    search_edit 초기화 + category_combo 전체 설정 + run_search()

  _update_status_bar():
    statusBar().showMessage() 로 데이터 범위·ADRG/TABLE 수·버전·면책 안내

  open_about_dialog():
    AboutDialog(parent, app_version, store).exec()

  _restore_settings():
    QSettings("KDRG", "KDRGRelationSearch") 에서 geometry/splitter/panelOpen 복원

  closeEvent():
    geometry/splitter/panelOpen QSettings에 저장

  ── CodeTableFrame 변경 ──
  · 3열: ["코드", "한글명", "영문명"]
  · display_name 파싱: "en / ko" 분리 → 별도 열에 표시
  · _populated = False (지연 채우기)
  · ensure_populated() 추가: 처음 펼칠 때만 populate_table() 호출
  · populate_table() 내 apply_filter 동작도 3열 검색으로 확장
